# Story B8 - Dashboard Shell

**Epic:** 3 - Dashboard  
**Status:** Done  
**Dependencies:** B1, A7  

## Story

As a public visitor, I want a responsive dashboard shell so that trend, sector, and map visualizations have a clear, polished layout.

## Acceptance Criteria

- Header and dashboard layout exist.
- Controls are scoped near the visualization they affect.
- Layout is responsive and has no horizontal overflow.
- Shared chart card state patterns exist.

## Architecture Context

- Use MUI components and theme tokens.
- Avoid global filter bar behavior.
